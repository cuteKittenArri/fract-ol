# fract-ol

## Description

fract-ol is a small fractal rendering program built in C as part of the 42 school curriculum. The goal of the project is to explore the basics of computer graphics — window management, pixel drawing, and event handling — while implementing the mathematics behind two classic fractals: the **Mandelbrot set** and the **Julia set**.

Both fractals are generated by iterating a complex function and coloring each pixel based on how quickly (or whether) the sequence escapes to infinity. The result is an infinitely detailed boundary that can be explored through zoom.

## Instructions

### Dependencies

- X11 / Xorg (`libX11`, `libXext`)
- Standard math library (`-lm`)

### Compilation

```bash
make
```

| Target   | Description                     |
|----------|---------------------------------|
| `all`    | Compile the project             |
| `clean`  | Remove object files             |
| `fclean` | Remove object files and binary  |
| `re`     | Full recompile from scratch     |

### Execution

```bash
# Mandelbrot set
./fractol mandelbrot

# Julia set — requires a complex constant as two decimal numbers
./fractol julia <real> <imaginary>

# Example
./fractol julia -0.4 0.6
```

If invalid or missing arguments are provided, the program prints usage instructions and exits.

### Controls

| Input        | Action      |
|--------------|-------------|
| Scroll Up    | Zoom in     |
| Scroll Down  | Zoom out    |
| `ESC`        | Quit        |
| Window close | Quit        |
